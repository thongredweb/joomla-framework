Method to return a Driver instance based on the given options.

getInstance(array $options) : \Joomla\Database\Driver
InheritedStatic

There are three global options and then the rest are specific to the database driver. The 'driver' option defines which Driver class is used for the connection -- the default is 'mysqli'. The 'database' option determines which database is to be used for the connection. The 'select' option determines whether the connector should automatically select the chosen database.

Instances are unique to the given options and new objects are only created when a unique options array is passed into the method. This ensures that we don't end up with unnecessary database connection resources.
